# MINERVA AI — Critical Mineral Intelligence Platform

AI-powered uranium and critical mineral supply chain intelligence.

## Features
- Full marketing site targeting 5 buyer personas
- Live AI Signal Analyzer powered by Claude
- Real-time uranium fuel cycle monitoring (Mining → Conversion → Enrichment → Fabrication)
- Supplier database and route intelligence

## Environment Variables Required
Set in Vercel project settings:
- `ANTHROPIC_API_KEY` — your Anthropic API key (sk-ant-...)

## Tech Stack
- Next.js 14 (App Router)
- React 18
- Anthropic Claude Sonnet 4
